# CLI ToDo
Консольное приложение для управления задачами (создание, чтение, обновление, удаление, отметка выполнения) с хранением в PostgreSQL.

## Функциональность:
Добавить задачу (name, desc), показать задачу по `id`, изменить `name` и/или `desc`, удалить по `id`, отметить задачу выполненной.

## Требования и структура
- Go 1.25.
- Postgres БД.
- Чистая архитектура с разделением на слои

## Sprint Review (Спринт 1)

### Статистика спринта
- **Длительность:** 2 недели
- **Запланировано:** 5 задач
- **Выполнено:** 4 задачи
- **Не выполнено:** 1 задача
- **Сумма story points:** 11 из 14 запланированных

### Завершенные задачи
1. **Спроектировать модель данных** - создан класс Task со всеми полями
2. **Выстроить базовую архитектуру** - реализована слоистая архитектура (Core, Infrastructure, Console)
3. **Имплементировать слой Repository** - готов интерфейс и реализация для работы с данными
4. **Имплементировать слой Service** - реализована бизнес-логика приложения

### Что было доставлено
Создана основа приложения со слоистой архитектурой. Все ключевые компоненты (модель, репозиторий, сервис) готовы и протестированы. Приложение можно собрать и запустить, но функционал команд еще не полностью интегрирован.

### Что не успели
**Реализовать точку входа и консольный интерфейс** - частично реализован главный цикл, но не все команды интегрированы с сервисом. Задача переносится в следующий спринт.
