# CLI ToDo
Консольное приложение для управления задачами (создание, чтение, обновление, удаление, отметка выполнения) с хранением в PostgreSQL.

## Функциональность:
Добавить задачу (name, desc), показать задачу по `id`, изменить `name` и/или `desc`, удалить по `id`, отметить задачу выполненной.

## Требования и структура
- Go 1.25.
- Postgres БД.
- Чистая архитектура с разделением на слои
